INTERNSHIP

Monet Networks Inc.

Monet platform provides a way to capture, track and analyze video content in real-time and at scale using non-verbal cues (NVCs) from online viewers or tradional on-line panels within select demographics. Brands or content creators are provided a dashboard to push their content to traditional on-line panels and create campaigns, or a community of incentivized users. Monet platform collects the users rich audio and video reactions at scale and presents real-time actionable intelligence on content effectiveness to brands and content creators based on emotional engagement metrics and their cross-correlation with user self reported data. Monet’s user side platform provides a method for users to organically find and rate different brands’ content, earning valuable rewards in return.

The Project

In the first few days I spend some time in getting familiar with the big database and several files of the project and implemented some basic tasks like updating emotions that has been captured by the face-reader api which was achieved by writing several sql queries and shell scripts in php. Then in the first phase I implemented the feature of analysing the text, like comments or open-ended questions asked in surveys, and get emotions out of it. Text is also get filtered by some pyhton scripts to detect the gibberishness in which a model of character to character transitions from bunch of text in English based on Markov Chain is built and trained to get probabilities of adjacent pairs of characters in a string. These probabilities signifies how often characters appear next to each other.

Broadening the Platform

Previously, in monet, a corporate can only create campaigns on videos and that are just youtube urls. I added the features of adding video campaigns from uploaded video files, image campaigns from both url and uploaded image files and text campaigns from both pasting text on textbox or uploaded text files. All these types of campaigns can be watched by users as well and for that their template files have been created with certain features in each of them like a countdown timer with play and pause in image campaigns, a custom text animation with speed, play and pause in text campaigns. Each type of campaigns uses wowza streaming engine, when browser's flash is enabled, to capture snapshots in real-time and when flash is disabled then it uses get-user-media feature of browsers to capture snapshots.

Gyro-Stabilized Plane

A composite RC plane which has self-balancing ability while flying in mid air. It can stabilize itself, whenever the external factors like high wind speed tries to alter the orientation of plane, by manipulating its main control units using arduino micro-processor.

Objective

Objective was to make a composite RC plane embedded with Arduino (micro-processor), which is stable even in high wind speed and used for maneuvering purposes. Developed a code written in Arduino IDE using PID (proportional-integral-derivative) controller feedback mechanism to find error in all three axes and then rectified. Accelerometer and gyro-meter were used to find orientation of plane during its flight and got success in achieving favorable statistical data of test flights.

The Game of Cops and Robbers on Graphs

The game of cops and robbers is played with a set of ‘cops’ and a ‘robber’ who occupy some vertices of a graph. Both sides have perfect information and they move alternately to adjacent vertices. The robber is captured if at least one of the cops occupies the same vertex as the robber.

Paragraph and Image

Studied theorems on upper and lower bounds on cop numbers required to capture robber after finite number of rounds. Examined more about the retraction of graphs and dismantlable graphs and their theorems.
